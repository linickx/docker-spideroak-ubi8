# Security

- Images are hosted on docker-hub, please check the publish date to understand how recent each image is:
    - https://hub.docker.com/r/linickx/spideroak-ubi8
    - https://hub.docker.com/r/linickx/spideroak-ubi10
    - https://hub.docker.com/r/linickx/spideroak-trixie
- A vulnerability scan report can be found in each `SCAN.md`
    - https://github.com/linickx/docker-spideroak/blob/master/SCAN.ubi8.md
    - https://github.com/linickx/docker-spideroak/blob/master/SCAN.ubi10.md
    - https://github.com/linickx/docker-spideroak/blob/master/SCAN.trixie.md
- A Software Bill of Materials can be found in each `SBOM.spdx.json` or via GitHub's [Dependency Graph](https://github.com/linickx/docker-spideroak/network/dependencies)
    - https://github.com/linickx/docker-spideroak/blob/master/SBOM.ubi8.spdx.json
    - https://github.com/linickx/docker-spideroak/blob/master/SBOM.ubi10.spdx.json
    - https://github.com/linickx/docker-spideroak/blob/master/SBOM.trixie.spdx.json
- To report a security issue, contact me via my website [linickx.com](https://www.linickx.com) or Mastodon [@linickx@infosec.exchange](https://infosec.exchange/@linickx)
